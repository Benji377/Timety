# Frontend

## Navbar
- Focus icon should be a steaming coffee mug
- Stats icon should be a pie chart diagram
- When going to settings, hide the navbar and instead add an arrow to go back to the home page to not mess up navigation

## Home page
- There should be a suggestion to add a task if there is currently no task. Clicking it navigates to the tasks list
- The circular chart showing the time of focus should navigate to the focus page upon clicked

## Fokus page
- No title needed
- Two modes: timer or stopwatch
  - When in timer mode, you can select the time like a time picker by clicking the time indicator
  - In stopwatch, you don't select the time
- Below the timer are three buttons: start, pause and stop
  - Not all three buttons are enabled by default
  - start: starts the time
  - pause: pauses the time
  - stop: stops the time and saves it as a focus session (if the focus timer was started from a task, it saves it to that)
- Remove ambient sounds, users should use an external app for that

## Tasks page
- At the top right there should be a calendar icon that opens a full calendar with dots for the tasks
- Add a divider below the today task and show all upcoming tasks
- When adding a new task, let the user add all info: location, due date, due time, reminders, etc...
- A user should be able to modify a task
- The tasks page must show all tasks, in a sorted manner, daily are separated for easier overview
- A task has a blue border when status is TODO, a red border when overdue, a yellow border when due today and a green border when done
- User can start a focus timer for a task

## General
- Make app more colorful and happy
- Use more vivid colors for the theme
- Add dark mode to the settings
- Allow local import and export
- Save settings in proper categories like Backup containing Google drive, export and import
